## Configuration
add config mongodb connection to `.env` file :

`MONGO_DB_HOST=127.0.0.1  
MONGO_DB_PORT=27017  
MONGO_DB_DATABASE={your_db_name}  
MONGO_DB_USERNAME={username}  
MONGO_DB_PASSWORD={password}`
